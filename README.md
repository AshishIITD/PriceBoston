# 🏘️ Boston Housing Price Prediction

A machine learning regression project for predicting house prices in Boston using classical and advanced ML algorithms. This project demonstrates the application of data science techniques in real estate price estimation.

## 📊 Overview
This project analyzes the Boston Housing dataset and builds predictive models to estimate house prices based on various features such as crime rate, number of rooms, age, etc. It's an excellent example of supervised learning and regression analysis.

## 🛠️ Technologies Used
- **Python** - Core language
- **Scikit-Learn** - Machine learning library
- **Pandas** - Data analysis and manipulation
- **NumPy** - Numerical computing
- **Matplotlib & Seaborn** - Data visualization
- **Jupyter Notebook** - Interactive development

## 📈 Dataset Features
- **Samples:** Boston Housing data
- **Features:** 13 predictive variables
- **Target:** House prices (MEDV)
- **Variables:** RM (rooms), LSTAT (lower status %), AGE, RAD (accessibility), etc.

## 🎯 Objectives
- Build accurate regression models for price prediction
- Perform exploratory data analysis (EDA)
- Compare multiple ML algorithms
- Evaluate model performance using standard metrics

## 🚀 Models Implemented
- Linear Regression
- Ridge Regression
- Lasso Regression
- Decision Trees
- Random Forest
- Gradient Boosting

## 📊 Evaluation Metrics
- Mean Squared Error (MSE)
- Root Mean Squared Error (RMSE)
- Mean Absolute Error (MAE)
- R² Score
